# web云桌面

HTML5 远程桌面解决方案，通过访问浏览器就能操作远程服务器。

### Demo
```bash
http://dcloud.stbui.com
```


### 安装依赖

```
npm install
```

### 启动服务

```
npm start
```

### 访问地址

```
http://127.0.0.1:8361
```

## pm2 管理
```
pm2 startOrReload pm2.json
```
