# web云桌面

HTML5 远程桌面解决方案，通过浏览器就能操作远程服务器，适用于Chrome、Firefox、IE9+等浏览器（浏览器需要支持HTML5）， 由于使
用 HTML5只要在一个服务器安装成功，你访问你的桌面就是访问一个 web 浏览器
